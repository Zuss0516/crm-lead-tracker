# CRM Lead Tracker

A lightweight, browser-based CRM built for small businesses to track leads, pipeline value, and follow-ups — no backend, no database, no setup required. Just open the file and start working.

---

## Live Demo

https://github.com/Zuss0516/crm-lead-tracker

---

## What it does

- Add, edit, and delete leads with a single click
- Track each lead through a clear pipeline: **New → Contacted → Qualified → Closed → Lost**
- See live summary stats: total leads, pipeline value, closed count, and overdue follow-ups
- Filter leads by status or search by name and service
- Overdue follow-up dates are automatically highlighted in red so nothing gets missed

## How to use it

**Option 1 — Run locally**

1. Download or clone this repo
2. Open `index.html` in any browser
3. That's it

**Option 2 — Deploy to GitHub Pages**

1. Fork or push this repo to your GitHub account
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. GitHub will give you a live link within a minute

## Who this is for

Any small business that needs a simple way to manage leads without paying for a full CRM platform. This is especially useful for:

- Auto detailing shops tracking PPF, ceramic coating, and tint inquiries
- Freelancers managing client pipelines
- Service businesses following up on quotes
